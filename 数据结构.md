## 快排算法：

```java
private static void quickSort(int[] arr, int left, int right) {
        if(left>=right)return;
        int i=left+1,j=right;//左右侧空位索引
        int value=arr[left];
        int vi=left;
        while(i<=j){
            if(arr[i]<value){
                swap(arr,i,vi);
                i++;
                vi++;
            }else if(arr[i]>value){
                swap(arr,i,j);
                j--;
            }else i++;
        }
        quickSort(arr,left,vi-1);
        quickSort(arr,j+1,right);//注意是j+1>=vi+1
    }
```
